# BrowserBox Webview Element API

`browserbox-api` is the public BrowserBox Embedder API package: a browser webview for the open web.
It provides the drop-in `<browserbox-webview>` element used to embed full BrowserBox sessions into custom web applications and rich UI shells.

Package target: `@browserbox/webview-element`.

## Scope

- `<browserbox-webview>` custom element
- Browser-side API surface and events
- Integration docs and examples
- Foundation for advanced integrations, including fully themed/reskinned browser experiences

## Documentation

- BrowserBox Embedder API documentation: https://win9-5.com/api/
