Overview

Hyper-Frame bypasses traditional iframe restrictions like X-Frame-Options: DENY and Content-Security-Policy by leveraging a secure remote browser isolation (RBI) backend. It allows developers to embed any website, interact with it programmatically, and capture its state in real-time.

  • Navigate: Control the remote browser's URL and history.
  • Automate: Click, type, and evaluate JavaScript within the frame.
  • Capture: Stream live frames or take high-quality screenshots.
  • Secure: Keep high-risk browsing activity isolated from your users' local environments.
